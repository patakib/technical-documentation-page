Introduction

Django is a backend framework for web development, based on Python programming language. Django is a fully open-source and free project, which you can even contribute to. The framework is a perfect choice for database-heavy web projects. Django's first release was in 2005, it is maintained by the Django Software Foundation.

Django comes with a built-in admininstrative site, which is a practical and easy feature to support CRUD (create, retrieve, update, delete) operations. It also has an ORM (Object-relational Mapper) that manages relational databases through "models", which are actually Python classes. Django has a simple web server for development purposes.

Its community is active and huge. There is a regular conference since 2008, called DjangoCon.
